Eval as in our old enemy eval(). This function is called on every single $_POST key/value pair after form submission. $On_Insert is an array of executable PHP code stored in the database. Figure out the rest of the misery yourself; I might jump off a cliff if I have to think through this code again. 
Function doPostEval($Key,$Value,$On_Insert)
{
    $Code = $On_Insert[$Key];
    $xxx = '';
    if ($Code <> '') {
        eval($Code);
        return($xxx);
    } else {
        return($Value);
    }
}
